# MusicRitmo App

## Настройка окружения

1. Клонируйте репозиторий с помощью

```bash
https://github.com/BenSerg/Music-Ritmo.git
```

2. Перейдите в директорию проекта

```bash
cd frontend
```

3. Установите зависимости

```bash
npm install
```

## Запуск приложения

1. Запуск приложения

Сборка

```bash
npm run build
```

Запуск

```bash
npm start
```

1.1 Запуск приложения в режиме разработки

```bash
npm run dev
```
